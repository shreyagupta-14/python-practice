# python-practice

This is my personal workspace for practicing Python concepts, testing scripts, and keeping my core programming skills sharp.

Instead of just watching tutorials passively, I use this repo to write out scratchpad scripts, break things locally, and figure out how different Python standard libraries work.

## What's in here?
* **Standard Modules:** Quick, isolated scripts to understand built-in Python tools (like `argparse` for CLI commands, `os`, `sys`, etc.).
* **Syntax & OOP Sandbox:** Small files testing functions, classes, and logic flow before putting them into actual projects.
* **Problem Solving:** Drafts and logic checks for daily problem-solving and SQL practice.

## Current Setup
* **Language:** Python 3.x
* **Environment:** VS Code / Terminal

*Note: This repository is intentionally a bit messy because it functions as my digital rough notebook. Finished, standalone projects live in their own dedicated repositories on my profile.*
